# 开题报告：基于图神经网络步长预测的三维 MMC 悬臂梁拓扑优化方法研究

## 一、课题名称

基于图神经网络步长预测的三维移动可变形组件悬臂梁拓扑优化方法研究

## 二、研究背景与意义

拓扑优化能够在给定设计域、载荷和约束条件下自动寻找高性能结构构型，是轻量化设计、增材制造、工程结构优化和智能设计中的重要技术。传统密度法拓扑优化依赖规则网格上的单元密度变量，表达能力强，但在三维问题中设计变量数量巨大，优化过程计算成本高，且结果常需要后处理才能转化为明确的几何边界。

移动可变形组件方法（Moving Morphable Component, MMC）以显式参数化组件描述结构。每个组件由中心、尺度和旋转角等少量变量控制，多个组件通过隐式拓扑描述函数组合形成结构边界。与纯密度法相比，MMC 方法具有设计变量少、几何边界清晰、适合后续 CAD/制造转换等优势。然而，在三维问题中，MMC 优化仍需要反复进行有限元分析和灵敏度计算，优化效率与初始组件布局、更新步长、几何可行性和连通性密切相关。

近年来，图神经网络（Graph Neural Network, GNN）在非欧几里得结构数据建模中表现出较强能力。MMC 结构天然可以表示为组件图：节点对应组件，边对应组件间几何关系、重叠关系和潜在传力关系。因此，可以利用 GNN 学习优化过程中的历史响应和局部结构关系，为传统优化器提供辅助决策。本课题拟研究一种物理约束下的 GNN 步长预测方法，使神经网络不直接替代有限元或灵敏度方向，而是在 MMA 给出的更新方向上预测安全有效的步长缩放系数，从而提高三维 MMC 悬臂梁优化的效率和稳定性。

## 三、研究问题

本课题围绕三维悬臂梁 MMC 拓扑优化中的以下问题展开：

1. 三维 MMC 组件在优化过程中容易出现旋转后实体越出设计域的问题，需要建立严格的几何可行域约束。
2. 完全随机分散的初始组件难以形成从固定端到载荷端的连续传力路径，容易导致优化结果断裂或停滞。
3. 传统 MMA 的固定步长或候选步长策略难以适应不同迭代阶段、不同载荷位置和不同组件状态。
4. 若直接用神经网络预测结构或替代物理求解，容易失去可靠性和可解释性。
5. 三维拓扑优化过程需要可视化真实隐式边界，而不仅是逐组件表面图，否则难以判断结构是否连续。

## 四、研究目标

本课题拟实现并验证一套面向三维 MMC 悬臂梁拓扑优化的“物理优化器 + 图神经网络步长预测”框架。具体目标如下：

1. 建立 `60 x 4 x 20` 三维悬臂梁设计域下的 MMC 参数化优化流程。
2. 实现旋转后 MMC 组件的严格设计域约束，保证组件实体不越界。
3. 构建面向 GNN 的 MMC 组件图数据集，包含节点特征、边特征、全局特征、候选步长响应和连通性诊断标签。
4. 训练物理偏置图注意力网络（GAT），预测图级和组件级步长缩放系数。
5. 引入 MAML 风格元学习，使模型能够适应不同载荷位置、不同初始布局和不同优化阶段。
6. 建立在线 FEM 验证、安全回退和 trust-bias 微调机制，保证神经网络预测不破坏优化可靠性。
7. 通过隐式边界等值面绘图展示真实 MMC 拓扑演化过程。

## 五、研究内容与技术路线

### 5.1 三维 MMC 悬臂梁优化模型

设计域设为：

```text
DL = 60.0, DW = 4.0, DH = 20.0
```

网格划分为：

```text
nelx = 60, nely = 4, nelz = 20
```

边界条件为左端面固定，右端面附近施加 `-Z` 方向集中力。结构由 24 个三维 MMC 组件描述，每个组件包含 9 个设计变量：

```text
x0, y0, z0, L1, L2, L3, alpha, beta, gamma
```

目标函数为最小柔度：

```text
min C = F^T U
```

约束为体积分数：

```text
volume_fraction <= 0.4
```

优化器采用 MMA，并保留有限元分析和解析灵敏度链路。

### 5.2 几何可行域约束

针对三维组件旋转后越界的问题，采用旋转后轴对齐包围盒（AABB）判断组件实体是否在设计域内。对每个组件计算：

```text
half_extent = abs(R) @ [L1, L2, L3]
```

并要求：

```text
center - half_extent >= 0
center + half_extent <= [DL, DW, DH]
```

当 MMA 或候选步长产生不可行参数时，先对组件尺度和中心进行投影，再进入 FEM 评估、图样本保存和过程图绘制。

### 5.3 悬臂梁初始组件布局策略

当前代码已支持 24 个随机分散组件的严格域内初始化。初步实验表明，完全随机分散的初始化通常不能形成连续传力路径，50 次迭代后仍可能呈现分散块状结构。因此，后续正式数据集将采用悬臂梁专用随机初始化策略：在固定端、载荷端和中间桥接区域设置基本覆盖，同时对组件中心、长度和角度施加随机扰动，以兼顾结构可行性与数据多样性。

### 5.4 GNN 图数据构建

每个优化迭代步导出一个 MMC 组件图。节点为 MMC 组件，边由几何距离、方向关系、重叠关系和接触评分建立。节点特征包括归一化中心、尺度、旋转角、组件体积、灵敏度相关统计量、载荷距离先验和应变能近似特征。全局特征包括迭代步、柔度、体积分数、目标体积分数、载荷位置和组件数量。

对每个 MMA 更新方向，程序评估 6 个候选步长：

```text
0.25, 0.5, 0.75, 1.0, 1.25, 1.5
```

每个候选均通过真实 FEM 计算柔度和体积分数，用于生成 `eta_label`、`eta_feasible_mask` 和 `response_targets`。若所有候选都违反体积约束，则记录失败标志，分类损失中屏蔽该样本。

### 5.5 物理偏置 GAT 步长预测

GAT 模型以 MMC 组件图为输入，输出图级步长预测和组件级步长预测。模型引入载荷距离先验作为注意力偏置，使靠近载荷路径的组件在消息传递中获得更高关注。为避免物理先验权重变为负数，注意力偏置采用非负约束：

```text
actual_alpha = 0.1 + 0.9 * sigmoid(raw_alpha)
```

预测步长限制在安全范围内：

```text
eta in [0.25, 1.5]
```

该设计保证神经网络只控制传统优化方向上的步长缩放，不替代 FEM、灵敏度或 MMA 方向。

### 5.6 元学习与在线安全控制

不同轨迹、载荷位置和初始组件布局可视为不同任务。MAML 风格训练按轨迹构造 task，在同一轨迹内划分 support/query 步骤，使模型获得少样本适应能力。

在线阶段使用如下安全机制：

1. GAT 预测组件级步长。
2. 将步长投影到安全范围。
3. 用 FEM 验证预测步长后的柔度和体积约束。
4. 若预测偏差超过阈值，触发 trust-bias 检查。
5. 对少数关键组件生成稀疏真实标签并在线微调。
6. 若神经网络预测不可靠，则回退到保守候选步长搜索。

### 5.7 隐式边界等值面可视化

为了获得接近论文图的连续红色拓扑效果，本课题不伪造圆柱杆件，而是从真实 MMC 全局隐式函数中提取 `phi = 0` 等值面。绘图流程为：

1. 在 `60 x 4 x 20` 设计域中采样全局 TDF。
2. 使用 KS 聚合得到全局隐式边界。
3. 通过 `skimage.measure.marching_cubes` 提取等值面。
4. 以白色背景、黑色坐标轴、固定刻度和红色光滑表面输出过程图。

每条轨迹保存初始、中间和最终三阶段过程图，便于分析拓扑演化和连通性。

## 六、实验方案

### 6.1 基础正确性验证

首先进行小规模 smoke test：

```bash
python -B generate_dataset.py --num-trajectories 1 --max-iter 5
```

验证内容包括：

- 图样本数量是否正确。
- 组件旋转后 AABB 是否均在设计域内。
- `eta_candidates.shape == (6,)`。
- 过程图和等值面图是否生成。
- FEM 求解是否无异常。

### 6.2 初始化策略对比

对比两类初始化：

1. 完全随机分散初始化。
2. 悬臂梁专用随机初始化。

评价指标包括：

- 初始是否存在固定端到载荷端的连通路径。
- `connected_to_load` 比例。
- `spanning_ratio`。
- 最终柔度。
- 体积分数约束满足情况。
- 迭代停滞次数。

当前随机分散初始化的 1 条 50 步实验显示，51 个图样本均未形成有效载荷连通路径，最终柔度约为 `5.99e3`，说明完全随机撒点不适合作为正式训练数据的默认策略。

### 6.3 数据集生成与划分

在初始化策略稳定后，逐步扩大数据规模：

```bash
python -B generate_dataset.py --num-trajectories 20 --max-iter 20
python -B generate_dataset.py --num-trajectories 100 --max-iter 20
python -B generate_dataset.py --num-trajectories 500 --max-iter 20
```

数据集按轨迹划分训练、验证和测试集，避免同一轨迹相邻迭代步泄漏到不同集合。

### 6.4 GAT 与 MAML 训练

离线训练比较以下模型：

1. 仅图级 eta 监督的 GAT。
2. 图级与组件级联合监督的 GAT。
3. 加入候选响应预测辅助任务的 GAT。
4. MAML 风格元学习预训练模型。

评价指标包括：

- eta 回归误差。
- eta 分类准确率。
- 候选响应预测误差。
- 失败样本识别能力。
- 新载荷位置上的泛化能力。

### 6.5 在线闭环验证

将训练后的 GAT 接入 MMC 主优化循环，比较：

1. 原始 MMA。
2. MMA + 固定候选步长搜索。
3. MMA + GAT 步长预测。
4. MMA + GAT + 在线微调 + 安全回退。

评价指标包括：

- 达到相同柔度所需 FEM 次数。
- 最终柔度。
- 体积分数约束违反次数。
- 连通性破坏次数。
- 优化停滞次数。
- 总运行时间。

## 七、预期创新点

1. 将三维 MMC 拓扑优化过程表示为组件图，并面向步长预测构建 GNN 数据集。
2. 提出“传统 MMA 方向 + GNN 步长缩放”的保守学习框架，在保留物理可靠性的前提下引入学习能力。
3. 在图注意力中引入载荷距离先验，并通过非负权重约束保证物理偏置方向合理。
4. 将候选步长 FEM 响应作为监督信号，使模型不仅学习标签，还学习局部响应曲线。
5. 将旋转组件 AABB 可行域约束和密度场连通性诊断纳入 MMC 数据生成流程。
6. 使用真实全局隐式边界等值面绘制三维 MMC 拓扑演化过程，避免用伪造几何连接替代优化结果。

## 十、风险与应对措施

风险一：随机初始化导致结构不连通。应对措施是引入悬臂梁专用随机初始化，在保证基本传力路径的基础上加入随机扰动。

风险二：三维 FEM 计算成本过高。应对措施是采用分阶段数据规模，从 `1 x 5`、`20 x 20` 逐步扩大到正式训练规模。

风险三：GNN 预测步长导致约束恶化。应对措施是所有在线预测必须经过 FEM 验证，并设置保守候选搜索回退。

风险四：组件级标签不够真实。应对措施是先使用图级标签广播打通链路，后续通过组件局部候选评估或关键组件稀疏标签生成真实组件级监督。

风险五：论文式绘图可能误导结构判断。应对措施是只从真实 TDF 提取等值面，不人为补洞、不伪造连接，并同步输出连接性诊断指标。

## 十二、核心论点

本课题的核心论点是：在三维 MMC 悬臂梁拓扑优化中，将 GNN 限定为传统优化方向上的步长预测器，可以在保留 FEM 和灵敏度可靠性的同时利用历史轨迹数据提升优化效率；而几何可行域约束、连通性诊断和隐式等值面可视化是该方法从调试代码走向可信科研实验的必要条件。
